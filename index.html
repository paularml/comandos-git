Comandos Git para integração com Github

# Configure o nome e e-mail que irão estar atrelados aos seus commits:
git config --global user.name "Seu nome"
git config --global user.email "seu-email@example.com"

# Adicione uma pasta .git oculta ao seu repositório local. Isso fará com que o Git passe a trackear suas atividades:
git init

# Download de um repostório remoto para seu ambiente local:
git clone url_ssh_ou_https

# Lista modificações nos arquivos (útil para ver o que precisa ir para o stage ou o que já foi):
git status

# Para adicionar um arquivo específico ao stage:
git add nome_do_arquivo

# Para adicionar todos os arquivos ao stage:
git add .

# Commitando as alterações:
git commit -m "Mensagem do commit"

# Envia as modificações para o repositório remoto:
git push origin nome_da_branch

# Adiciona todas as mudanças feitas em arquivos trackeados ao stage e commita (não adiciona novos arquivos no stage):
git commit -am "Mensaem do commit"

# Puxa alterações de um repositório remoto para seu repositório local:
git pull origin nome_da_branch

Histórico de commits

# Lista histórico de commits:
git log

# Listar o histórico de commits (de forma mais resumida):
git log-oneline

# Mostrar alterações entre dois commits:
git diff commit1_ID commit2_ID

# Remove o arquivo do stage:
git reset nome_do_arquivo

Branch

# Lista todas as branchs locais:
git branch

# Criando uma branch
git branch nome_da_branch

# Mudando para uma branch:
git checkout nome_da_branch

# Criar uma nova branch e mudar para ela:
git checkout -b nome_da_branch

# Mesclar (mergear) duas branchs (a branch em que você está será mergeada com outra):
git merge nome_da_branch

# Excluir uma branch local:
git branch -d nome_da_branch

# Excluir uma branch remota:
git push origin --delete nome_da_branch

Revertendo (CUIDADO!)

# Remove o arquivo do stage:
git reset nome_do_arquivo

# Remover um arquivo da área de trabalho apenas:
git rm-cached nome_do_arquivo

# Remover um arquivo da área de trabalho e da área de preparação (stage):
git rm nome_do_arquivo

# Criar um novo commit, revertendo as alterações de um commit especificado::
git revert commit_ID

# Voltar para um commit anterior e excluir todos os commits à frente dele:
git reset commit_ID

# Permite excluir e/ou alterar o número de commits específicados (nesse caso 3):
git rebase -i Head~3

Extras

# Conectando seu repositório local a um repositório remoto:
git remote add origin url_ssh_ou_https

# Para ver todas as conexões remotas:
git remote

# Remover uma conexão::
git remote remove url_ssh_ou_https_da_conexão

# Pegar todas as branches do repositório remoto sem mergear:
git fetch

# Pegar uma branch específica do repostório remoto sem mergear:
git fetch nome_da_branch
